Plane Mirror Angle Of Incidence Visualization

Project Description

The Plane Mirror Angle Of Incidence Visualization is a Python project that allows users to visualize the angle of incidence on a plane mirror. The project presents an eye, a generic object, and a tall mirror that can be dragged and dropped anywhere on the screen. Once the user drops an object, the project displays lines that show the angle of incidence and a phantom version of the generic object behind the mirror where the eye would see the optical illusion. The project also shows light as lines that starts from the generic object and go in the direction of the eyes.

Requirements

  • Python 3.7 or higher
  • PySide6

Installation

  1. Clone the repository
  2. Install the required packages using the following command in the terminal:

pip install PySide6

  1. Run the following command to start the program:

python main.py

Usage

  • Drag and drop the eye, generic object, and mirror anywhere on the screen.
  • The project displays lines that show the angle of incidence and a phantom version of the generic object behind the mirror where the eye would see the optical illusion.
  • Light that starts from the generic object and goes in the direction of the eyes is also shown.

Project Structure

  • main.py: The main file that contains the main function and initializes the QApplication object, the MainWidget object, and shows the MainWidget object.
  • dragable.py: A Python file that contains the Dragable class, which is used to create dragable QLabel objects.
  • README.md: The file that contains the README for the project.
  • eye.png: An image file that contains the image of an eye.
  • thing.png: An image file that contains the image of a generic object.
  • mirror.png: An image file that contains the image of a tall mirror.
